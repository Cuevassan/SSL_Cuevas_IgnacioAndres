# 00-PrimerProgramaC

Este es el primer trabajo práctico (TP0) individual de la materia. El objetivo principal es validar que el entorno de desarrollo (compilador y herramientas de control de versiones) esté correctamente configurado. 

## Descripción
El proyecto consiste en un programa simple en lenguaje C que imprime un mensaje de bienvenida por consola para verificar el funcionamiento de `gcc`. 

## Instrucciones de Uso

### 1. Compilación
Asegúrate de estar dentro de este directorio en tu terminal (Git Bash recomendado en Windows) y ejecuta:

gcc archivo.c -o archivo.exe
